Introduction of HTML

What is HTML?

HyperText Markup Langauge, HTML is the standard document format for web pages which is supported by all browsers, also it is the Pre-definded tag based Langauge.

What is HyperText?

Hypertext is a method of structuring and linking digital documents.
Hypertext is text displayed on a computer display or other electronic devices with references (hyperlinks) to other text that the reader can immediately access.

What is Markup Langauge?

A markup language is a set of rules that defines how text and images should appear in a digital document. It's a text-encoding system that uses symbols or tags to control the structure, formatting, and relationships between parts of a text document.
A markup language is a computer language that uses tags to define elements within a document.
